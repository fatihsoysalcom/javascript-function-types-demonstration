# JavaScript Function Types Demonstration

This example demonstrates various types of JavaScript functions, including traditional function declarations, function expressions, modern arrow functions, immediately invoked function expressions (IIFEs), and object methods. It showcases their syntax and basic usage, with output displayed on the web page and in the browser's developer console. This helps illustrate the different ways functions can be defined and utilized in JavaScript applications.

## Language

`html`

## How to Run

1. Save the code as `index.html`.
2. Open `index.html` in any modern web browser.
3. Open the browser's developer console (usually by pressing F12) to see all logged output.
